Images can improve the desing and the appearance of a web page.

HTML Images Syntax

The HTML <img> tag us used to embed an image in web page.

Images are not technically inserted into a web page; imaged are linked to web pages. the <img> tag created a holding space for the referenced image.

The <img> tag has two required attributes:

  • src - specifies the path to the image
  • alt - specifies an alternate text for the image



The src attribute

the required src attributes specifies the path (URL) to the image

Note: When a web page loads at that gets the image from a web server and inserts it into the page. Therefore, make sure that the image actually stays in the same spot in relation to the web.



The alt attribute

the required alt attribute provides an alternate text for an image, if the user for some reason cannot view it (because of slow connection, an error in the src attribute, or if the user uses a screen reader)
