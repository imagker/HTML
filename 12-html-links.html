HTML links

Links are found in nearly all web paes. Links allow users to clik their way from page to page.



HTML Links-Hyperlinks

HTML links are hyperlinks.

you can click on a link and jump to another document.

when you move the mouse over a link, the mouse arrow will turn into a little hand.

Note: a link does not have to be a text. it can be any HTML element



HTML links-Syntax

the HTML <a> tag defines a hyperlink. It has the following syntax:

<a href="url">link text</a>

the most important attribute of the <a> element is the <href> attribute, which inidicates the link's destination.

by default, links will appear as follows in all browsers:

  • An unvisited link is underlines and blue
  • a visited link is underlined and purple
  • an active link is underlined and red

Tip: links can of course be style styled with CSS, to get another look!



HTML links - the target attribute

by default, the linked page will be displayed in the current browser window. to change this, you must specify another target for the link.

the target attribute specified where to open the linked documnet.

the target attribute can have one of the following values

  • _self - default opens the documnet in the same window/tab as it was clicked
    _self-links
  • _blank - opens the doc in a new window or tab
    _blank-links
  • _parent - opens the doc in the parent frame
    _parent-links
  • _top - opnes the doc in the full body of the window
    _top-links



Absolute UTLs vs. Realative URLs

Absolute URL is a full web adress in the hrefattribute

a local link ( a link to a page within the same website) is specified with a relative URL (without the "http://www" part)



HTML links - image , email adress, button as a link

to use an image as a link, just put the <img> tag inside the <a> tag


use mailto: inside the href attribute to create a link that opens the user's email program:
<a href="mailto:chicken@example.com">


to use an HTML button as a link, you have to add some JS code.

JS allows you to specify what happens at certain events, such as a click of a button.
