HTML elements

An HTML element is defined by a start tag, some content, and an end tag.

HTML elements

The HTML element is everyting from start tag to the end tag:

<tagname>Contents goes here...</tagname>

Stat tag Element content End tag
<h1> My first heading </h1>
<p> My first paragraph </p>
<br> none none

note: some HTML elemtes like <br> have no content. these elem are called empty elem. empty elem do not have an end tag.
