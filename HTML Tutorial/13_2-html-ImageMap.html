HTML Image Maps

With HTML image maps, you can create clickable areas on an image

Image Maps

The HTML <map> tag defines an image map. An image map is an image with clickable areas. The areas defined with one or more <area> tags.

Try click on the apple & banana & background of the image below :

Using Image Map

1. The image is inserted using the <img> tag. The only difference from other images is that you must add a usemap attribute :

<img src="apple_banana.jpg" alt="tabemono" usemap="#tabemono">

usemap value starts with a hash tag # followed by the name of the image map, and is used to create a realationship between the image and the image map.


2. Create img map by adding a <map> element.

the <map> element is used to create an img map, and is linked to the image by using the required name attribue:

<map name="tabemono">


3. finally add the clickable areas. a clickable area is defines using an <area> element.

<area shape="rect" coords="34, 44, 270, 350" href="computer.htm">



Map attributes

  • name - the name attribute gives the map a name so that it can be referenced. The attribute must be present and must have a non-empty value with no space characters. Also attribute must not be equal to the value of the name attribute of another <map> element in the same document. if the id attribute is also specified, both attributes must have the same value.

coords

the coords attribute details of the shape attribute in size, shape, and placement of an <area>

  • rect - the value is x1,y1,x2,y2. value specifies the coordinates of the top-left and bottom-right corner of the rectangle.

  • circle - the value is x,y,radius. value specified the coordinates of the circle center and the radius

  • poly - the value is x1,y1,x2,y2,...,xn,yn. value specifies the coordinates of the edges of the polygon. if the first and last coordinate pairs are not the same, the browser will add the last coordinate pair to close the polygon.

  • default - defines the enitre region
