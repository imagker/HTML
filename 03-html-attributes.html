HTML attributes

HTML attributes provide additional information about HTML elem.

HTML attributes

  • all HTML elem can hav attributes
  • attributes provide additional information about elements
  • attributes are always specified in the start tag
  • attributes usually come in name/vaule pairs like: name="value"

href attribute

the <a>(anchor) tag defines a hyperlink. the href(hypertext reference) attributes specifies the URL of the page the link goes to.


the src attrib

the <img> tag is used to embed image in an HTML page. the src(source attribute) attributes specifies the path to image to be displapyed:

<img src="img_chicken.jpg">

there are two ways to specify the URL in the src attribute

  1. absolute URL

    2. Links to an external image that is hosted on another website.
    Ex) src="https://www.w3schools.com/images/img_chicken.jpg"

    Notes: External images might be under copyright. also you cant contorl external images; it can suddenly be removed or changed.

  2. relative URL

    3. Links to an image that is hosted within the website. Here, the URL does not include the domain name. If the URL begins without slash it will be relative to the current page.
    Ex) src="img_chicken.jpg"
    If the URL begins with a slash, it will be relative to the domain.
    Ex)src="/images/img_chicken.jpg"

    Tip: It is almost always best to use realtive URLs. they woll not break if you cange domain.

the required alt attribute for the <img> tag specifies an alternate text for an image.

<img src="gamja_jpg" alt="this si gamja">

the style attribute

the style attribute is used to add style to an elem, such as color, font, size, and more.

<p style="color:red";>yo chicken</p>


the lang attribute

you should always include the lang attribute inside the html tag, to declare the lang of the Web page. this is meant to assist search engines and browsers.

country codes can also be added to the lang code in the lang attribute.

<html lang="en"> <html lang="en-US">


title attribute

the title attribute defines some extra information about an element. the value of the title attribute will be displayed as a tooltip when you mouse over the elem.

<p title="i'm a tooltip">this is a paragraph<p>


we suggest: always use lowercase attributes

the HTML standard does not require lowercase attributes names. however W3C recommends lowercase attributes in HTML, and demands lowercase attributes for stricter doc types like XHTML


we suggest: always quote attribute values

the HTML standard does not require quotes aroud attribute valuse. however, W3C recommends quotes in HTML, and demands quotes for stricter doc types like XHTML
