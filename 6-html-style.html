HTML styles

the HTML style attribute is used to add style to an elem, such as color, font, size, and more

the HTML style attribute

setting the style of an HTML elem, can be done with the style attribute.

the html style attribute has the foloowing syntax:

<tagname style="property:value;">

the property is a CSS property. the value is a CSS value

Background Color

the CSS background-color propety defines the background color for an HTML elem.

Text Color

the CSS color property defines the text color for an HTML elem.

Fonts

the CSS font-family property defines the font to be used for an HTML.

Text Size

the CSS text-size property defines the text size for an HTML elem.

Text Aligment

the CSS text-align property defines the horizontal text aligment for an HTML elem.
