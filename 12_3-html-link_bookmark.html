html-bookmarks

HTML links - create bookmarks

HTML links can be used to create bookmarks, so that readers can jump to specific parts of a web page.


Create a Bookmark in HTML

JUMP TO HOW TO


to create a bookmark - first create the bookmark, then add a link to it.

when the link is clicked, the page will scroll down or up to the location with the bookmark 





































































            
EX)
 1. use the id attribute to create a bookmark:
            <h2 id="C4">Chap.4</h2>

            then, add a link to the bookmark (jump to chp.4), from within the same page: 
            <a href="#C4">JUMP to Chap.4</a>

            you an also add a link to a bookmark on another page:
            <a href="html_demo.html#C4">JUMP to Chap.4</a>
