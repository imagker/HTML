HTML Quotation and Citation Elements

in this chapter we will go through the <blockquote>, <q>, <abbr>, <address>, <cite>, and <bdo> HTML elements.

HTML <blockquote> for Quotations

( this paragraph is written in blockquote )

the HTML <blockquote> element defines a section that is quoted from another source.

browsers usually indent <blockquote> elements

HTML <q> for short quotations

( this paragraph is written in quotation )

the HTML <q> tag defines a short quotation.

browsers normally insert quotation marks around the quotation.

HTML <abbr> for abbreviations

the HTML <abbr> tag defines an abbreviation or an acronym, like "HTML","CSS","Mr.","Dr.","ATM".

making abbreviations can give useful informaion to browsers, translation system and search-engines.

Tip: use the global title attribute to show the discription for the abbreviation/acronym when you mouse over the elem.

HTML <address> for contact information

the HTML <address> tag defines the contact information for the author/owner of a document or an article.

the contact info can be an email adress, URL, physical address, phone#, social media handel, etc.

the text in the <address> elem usually renders in italic, and browsers will always add a line break before and after the <address> elem.

HTML <cite> for work title

the HTML <cite> tag defines the title of a creative work (e.g. a book, a peom, a song, a movie, etc.)

Note: a person's name is not the title of a work.

the text in the <cite> elem usually render in italic

HTML <bdo> for bi-directional override

BDO stands for Bi-Directional Override.

the HTML <bdo> tag is used to override the current text direction:

like this following text will be written from right to left by <bdo dir="rtl">

hi my name is kim following text will be written from left to right by <bdo dir="ltr">
