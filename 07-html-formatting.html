HTML formatting

HTML contains several elem for defining text with a special meaning.

HTML formattting elem

formatting elem were desinged to display special types of text :

  • <b> - bold text
  • <strong> - important text
  • <i> - italic text
  • <em> - emphasized text
  • <mark> - marked text
  • <small> - smaller text
  • <del> - deleted text
  • <ins> - inserted text
  • <sub> - subscript text
  • <sup> - superscript text
